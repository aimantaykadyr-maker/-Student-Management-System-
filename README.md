
# 🖥️  Student Management System

**Knight of Silicon**

ArchRPG — бұл кэш-жад архитектурасын оқуды интерактивті әрі қызықты форматқа айналдыратын білім беру платформасы. Жоба пайдаланушыға тапсырмалар орындау, симуляция көру және алгоритмдерді практика арқылы түсінуге мүмкіндік береді. Сонымен қатар, қолданушы код жазып, оны бірден тексеріп, нәтижесін көре алады.


# 🚀 Негізгі мүмкіндіктер

**Интерактивті симулятор:**

* Адрес немесе мән енгізу арқылы нәтиже көру
* HIT / MISS анықтау
* Кэш күйін бақылау
* Қадамдық түрде кэш өзгерісін түсіну мүмкіндігі
* Қарапайым визуалды логика арқылы оқу процесін жеңілдету

**LRU алгоритмі:**

* Кэш толғанда ескі элементті өшіру
* LRU/FIFO логикасының базалық симуляциясы
* Элементтердің орын ауыстыру принципін түсіну
* Кэш тиімділігін практика арқылы көру

**AI-көмекші:**

* Тапсырма кезінде hint (кеңес) береді
* Қарапайым mock жүйе
* Қолданушы қателескенде бағыт береді
* Болашақта толық AI интеграцияға дайын

**Геймификация:**

* Python және C++ тапсырмалары
* Таймер (шешу уақыты)
* Дұрыс/бұрыс тексеру
* Прогресс сақтау
* Әр тапсырма — мини-квест ретінде
* Қолданушының жеке оқу процесін бақылау

**Python орындау:**

* Pyodide арқылы браузерде код тексеру
* Python кодты серверсіз орындау
* Айнымалылар алдын ала берілген (a, b, N, x)
* Нәтижені автоматты түрде тексеру


# 🛠 Технологиялық стек

**Frontend:**

* HTML5
* CSS3 (анимациялар, градиенттер, UI эффекттер)
* JavaScript (Vanilla)

**Логика:**

* Task system
* Cache simulation
* Timer
* Task validation (мәтін және код арқылы)

**Python:**

* Pyodide
* Браузер ішінде Python орындау
* Async execution (runPythonAsync)

**Storage:**

* LocalStorage
* Пайдаланушы деректері
* Тапсырма жауаптары
* Notes және visits сақтау

**AI (болашақта):**

* Gemini API
* Интерактивті оқыту көмекшісі
