# GitHub CLI (gh) Архитектурасы және Құрылымы

## Мақсаты

Бұл құжат `cli/cli` GitHub репозиторийінің ішкі құрылымын, компоненттерін және өндіріс ағындарын түсіндіреді.

## 1. Жалпы Сипаттама

**GitHub CLI (gh)** - бұл GitHub-тың ресми командалық жолы құралы. Ол developers-тің pull request-тер, issue-тер, fork-тар және басқа GitHub операцияларын терминалдан тікелей орындауына мүмкіндік береді.

## 2. Репозиторий Құрылымы

### Негізгі Папкалар

| Папка | Мақсаты |
|-------|---------|
| **cmd/** | Барлық CLI командаларының анықтамасы және реализациясы |
| **internal/** | Ішкі модульдер: API интеграциясы, аутентификация, кодспейс |
| **pkg/** | Қайта пайдаланатын пакеттер: utilities, форматтеу, браузер интеграциясы |
| **api/** | GitHub REST және GraphQL API клиенті |
| **git/** | Git операцияларының ор��ндалуы |
| **test/** | Unit тестілер |
| **acceptance/** | Интеграция және E2E тестілер |
| **docs/** | Разработка үшін ережелер және құжаттама |
| **.github/** | GitHub Actions CI/CD конфигурациялары |

## 3. Өндіріс Ағындары

### 3.1 Негізгі Команда Өндіріс Ағындары

```
Пользователь командасын енгізеді
    ↓
cmd/ - Командалық парсер
    ↓
internal/gh/ - Команда логикасы
    ↓
api/ - GitHub API қосылысы
    ↓
GitHub Сервері
    ↓
Нәтиже пользователькіге қайтарылады
```

### 3.2 Аутентификация Ағындары

```
gh auth login
    ↓
internal/auth/ - Аутентификация логикасы
    ↓
OAuth 2.0 ағындысы
    ↓
Token сохранялася (~/.config/gh/)
    ↓
Келесі операцияларда пайдаланылады
```

## 4. Ынамдау Ережелері

### 4.1 Қата Өңдеу

**Дұрыс түрі:**
```go
if err != nil {
    return fmt.Errorf("операцияны орындау сәтсіз болды: %w", err)
}
```

### 4.2 Pointer Тексеру

```go
// ✅ Сәтті қата өңдеу
if config == nil {
    return fmt.Errorf("конфигурация жүктелінген жоқ")
}
```

## 5. Build және Release

```bash
# Қалпыға түсіру
make build

# Барлық платформалар үшін
make dist

# Testing
make test
```

## 6. Қорытынды

`cli/cli` репозиторийі жақсы құрылымдалған Go жобасы болып табылады құнды ынамдау ережелері және толық тестілеу жүйесімен.
